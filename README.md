# Autonomous Research Analyst Agent

An end-to-end autonomous research agent that:
- Plans research queries
- Ingests real-world documents
- Uses RAG (FAISS) for grounding
- Self-evaluates confidence
- Generates consultant-style analytical reports

## Tech Stack
- Python 3.10
- LangChain
- OpenAI
- FAISS
